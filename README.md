# Full-Stack Task Manager

A full-stack task management application built with **React (Frontend)** and **FastAPI (Backend)**.

## 🚀 Features

- Create tasks
- Toggle task completion
- Delete tasks
- REST API backend
- CORS configured
- Full CRUD functionality
- Clean Git workflow

---

## 🛠 Tech Stack

Frontend:
- React
- JavaScript
- Fetch API

Backend:
- FastAPI
- Python
- Uvicorn

Tools:
- Git
- GitHub

## ▶️ How to Run Locally

### Backend

```bash
cd backend
.\.venv\Scripts\Activate.ps1
python -m uvicorn main:app --reload
```

Backend runs on:
```
http://127.0.0.1:8000
```

---

### Frontend

```bash
cd frontend
npm start
```

Frontend runs on:
```
http://localhost:3000
```
